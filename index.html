Research

My research focus on object detection and network compression, including in small/tiny/occluded object detection as well as model quantization, pruning and distillation. My research uses techniques from classical optimization, feature engineering and sometimes takes inspiration from cognitive science.

Confidence-driven Bounding Box Localization for Small Object Detection
Huixin Sun, Baochang Zhang, Yanjing Li, Xianbin Cao
Under Review, 2023
code / arXiv

We present a novel Confidence-driven Bounding Box Localization (C-BBL) method to rectify the gradient distortion phenomenon in small object detection.

MAFormer: A Transformer Network with Multi-scale Attention Fusion for Visual Recognition
Huixin Sun, Yunhao Wang, Xiaodi Wang, Bin Zhang, Chao Li, Ying Xin, Baochang Zhang, Errui Ding Shumin Han
Under Review, 2023
code / arXiv

We introduce Multi-scale Attention Fusion into transformer (MAFormer), which explores local aggregation and global feature extraction in a dual-stream framework for visual recognition.
